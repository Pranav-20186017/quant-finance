Program Overview:
This program is designed to process financial data and run machine learning models on it. The main function uses argparse to handle command-line arguments, allowing users to execute specific tasks like training models or generating charts.

Prerequisites:
- Python 3.11 installed
- Required Python libraries: argparse, pandas, yfinance, sklearn, Surampudi_Pranav_module1, Surampudi_Pranav_module4, Surampudi_Pranav_module5, Surampudi_Pranav_module6, Surampudi_Pranav_module7.

Installation:
1. Ensure Python 3.11 is installed on your system.
2. Install the required libraries using pip:
   pip install pandas yfinance scikit-learn
3. Place the Surampudi_Pranav modules in the same directory as the main program.

Usage:
To run the program, use the following command structure in the terminal or command prompt:
python <Surampudi_Pranav_project2.py> [command]

Commands:
- train_su: Trains models for the small universe dataset. This command reads a 'tickers.csv' file, processes the data, and trains both Support Vector Machine (SVM) and Extra Trees Classifier models. It outputs a CSV file Metrics_Small_Universe.csv containing model metrics.
- train_bu: Trains models for the broad universe dataset using Surampudi_Pranav_module7.
- chart_su_SVM: Generates charts based on the SVM model for the small universe dataset using Surampudi_Pranav_module4.
- chart_su_EXT: Generates charts based on the Extra Trees Classifier for the small universe dataset using Surampudi_Pranav_module5.
- chart_bu_SVM: Generates charts based on the SVM model for the broad universe dataset using Surampudi_Pranav_module6.

Examples:
- To train models for the small universe dataset:
  python Surampudi_Pranav_project2.py train_su
- To generate charts for the SVM model in the small universe dataset:
  python Surampudi_Pranav_project2.py chart_su_SVM
- To geenerate charts for the EXT Mmodel in the small universe dataset:
  python Surampudi_Pranav_project2.py chart_su_EXT


Notes:
- Ensure the 'tickers.csv' file is in the correct format and present in the same directory for the train_su command.
- The program's output files and charts will be saved in the same directory as the main program.
